# Area-Calculator


A simple, interactive Python script to calculate the area of different geometric shapes: Triangles, Squares, Circles, and Rectangles.


* **Triangle:** Calculates area using the base and height ($Area = \frac{1}{2} \times \text{base} \times \text{height}$).
* **Square:** Calculates area using the length of a side ($Area = \text{side}^2$).
* **Circle:** Calculates area using the radius ($Area = \pi r^2$).
* **Rectangle:** Calculates area using length and width ($Area = \text{length} \times \text{width}$).


To run this script, you only need Python 3.x installed on your machine. No external libraries are required.


1. Clone or download this repository to your local machine.
2. Open your terminal or command prompt and navigate to the project directory.
3. Run the script using the following command:

```bash
python area_calculator.py
